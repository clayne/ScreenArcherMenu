# ScreenArcherMenu

WIP mod for fallout 4 to aid in taking screenshots.

My modding discord for downloads and support https://discord.gg/Wkp8HDM4GP

Current features:
- Pose Adjustments
- Pose Exporter
- Play Idle
- Face morph sliders
- Eye Direction Control

Planned features:
- Save camera position
- Light positioning
- Mesh swaps
